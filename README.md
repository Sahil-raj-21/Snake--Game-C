# Snake Game in C

A console-based Snake Game developed in C/C++.

## Features
- Player Login System
- Multiple Levels
- Dynamic Obstacles
- Score Tracking
- High Score Storage
- Pause/Resume Functionality

## Technologies Used
- C/C++
- Windows Console
- File Handling

## How to Run

Compile:

g++ "Snake Game In C.cpp" -o SnakeGame.exe

Run:

SnakeGame.exe
